# A/B Test and EDA Case Study

Bu repo, bir mobil oyun veri seti üzerinde yaptığım A/B test modellemesi ve EDA çalışmalarına ait SQL sorgularını içeriyor.  
Analizleri BigQuery üzerinde gerçekleştirdim, sonuçları Google Sheets’te raporladım.

## Yapı

- **task1_ab_test/**  
  A/B testindeki alt maddeler için kullanılan SQL sorguları.

- **task2_eda/**  
  EDA tarafındaki temel dağılım, gelir ve oturum analizleri için kullanılan SQL sorguları.

- **reports/**  
  Google Sheets üzerinde hazırlanan özet sonuçların dokümanlaştırılmış halleri.

## Not

Veri seti ve firma bilgileri gizlilik nedeniyle repoda yer almıyor.  
Bu repo sadece analiz yaklaşımını ve sorguların yapısını göstermek amacıyla hazırlanmıştır.
